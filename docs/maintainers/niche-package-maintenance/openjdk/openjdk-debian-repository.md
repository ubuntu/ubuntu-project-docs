# OpenJDK Debian repository

OpenJDK packaging code is maintained on salsa[1].
The packaging code uses a separate branch per OpenJDK release.

Please tag the merge requests with the intended branches, e.g.
`[openjdk-11, ..., master]` to indicate that the merge request should be applied to all supported OpenJDK branches.

[1] https://salsa.debian.org/openjdk-team/openjdk

# Update the bug report

For regular bug fixes and merges, adding a comment about your progress is
typically all you'll need. You might provide some links to your PPA if you'd
like to get people to test your fix, or if you want to provide the fix to the
user-base swiftly.


## The SRU bug templates

If the bug fix targets the stable release, it must follow the {ref}`the SRU bug template <reference-sru-bug-template>`. Please see {ref}`Perform a standard SRU <howto-perform-standard-sru>` for the full overview of the
process.
